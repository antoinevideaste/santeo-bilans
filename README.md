# SANTEO · Générateur de bilans posturaux

Outil de génération de bilans posturaux personnalisés pour **PAYET Guillaume, Ostéopathe D.O**

## Installation locale

```bash
npm install
npm run dev
```

## Déploiement Vercel

1. Push ce dépôt sur GitHub
2. Connecter le repo sur [vercel.com](https://vercel.com)
3. Framework preset : **Vite**
4. Deploy → URL publique générée automatiquement

## Stack

- React 18 + Vite
- PWA (installable sur iPhone / Android / iPad)
- Génération PDF via impression navigateur
